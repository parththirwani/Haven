

Overview
This module exports two React components designed for a Next.js application with a dark-themed UI. It integrates Framer Motion for entrance animations and Lucide React for iconography.

CTASection Component
This component renders a call-to-action area intended to convert visitors into users. It features a scroll-triggered animation that fades in content when the section enters the viewport. Key elements include a floating lock icon, a headline emphasizing privacy, value propositions regarding cost and data privacy, and a styled button linking to the signup route.

Footer Component
This component renders the site footer. It displays the application branding (Haven), navigation links for authentication and the GitHub repository, and a dynamic copyright notice. It also includes a footer note highlighting security features like zero-knowledge architecture and end-to-end encryption.

Technical Details
- Framework: Next.js (Client Component)
- Styling: Tailwind CSS
- Animation: Framer Motion (useInView hook)
- Icons: Lucide React
- Theme: Dark mode with indigo accents
